# Momentum Strategy Vault / 动量策略金库

> AI 判断 BTC 方向，自动买入对应方向二元期权，止盈止损自动执行

## 解决什么问题

大多数链上预测市场参与者凭直觉交易，缺乏系统化的策略执行工具。动量策略金库通过 AI 模型分析 BTC 价格动量，自动在 Predict 市场买入对应方向的二元期权，配合自动止盈止损，帮助用户执行纪律化的趋势跟踪策略。

## 核心功能

- AI 模型每周期分析 BTC 价格动量（RSI、MACD、布林带等指标）
- 自动通过 predict::mint 买入看涨或看跌二元期权
- 可配置止盈止损：达到目标利润自动 redeem，达到止损自动平仓
- 策略回测面板：展示历史信号和模拟收益

## 使用的 DeepBook / Sui 能力

| 能力 | 用途 |
|------|------|
| predict::mint | 根据信号买入方向性二元期权 |
| predict::redeem_permissionless | 止盈/到期后自动赎回 |
| PredictManager | 统一管理金库的所有期权仓位 |
| OracleSVI | 辅助判断当前波动率是否适合进场 |

## 实现方案

### 智能合约 (Move)
- `momentum_vault` 模块：用户存款、份额铸造/销毁
- `signal_executor` 模块：接收链下信号、执行 mint 操作
- 关键数据结构：`VaultState`（总资产、持仓列表、策略参数）、`Signal`（方向、置信度、时间戳）

### Bot / Keeper / 服务端
- AI 信号引擎：Python 服务，每 5 分钟生成交易信号
- 信号提交器：将信号通过预言机或管理员接口提交到合约
- 止盈止损监控：实时追踪持仓盈亏，触发自动平仓
- 回测框架：基于历史预言机数据验证策略

### 前端 / 后端
- 技术栈：Next.js + Recharts + Python FastAPI
- 信号仪表盘：当前信号、历史信号准确率
- 收益追踪：BTC 计价净值曲线、夏普比率
- 策略参数配置页面

## 技术架构

Python AI 服务每 5 分钟分析 BTC 数据 → 生成信号 → 通过后端提交到合约 → 合约验证管理员签名后执行 predict::mint → Keeper 监控止盈止损条件 → 触发时提交 redeem PTB。前端展示信号流和收益曲线。
